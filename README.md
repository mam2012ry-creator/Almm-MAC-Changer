# 🛡️ Almm-MAC-Changer
A professional Python tool designed for Ethical Hackers to change network interface MAC addresses quickly and efficiently.

## 📸 Tool Preview
![Almm Tool Preview](preview.png)

## 🚀 Features
* **Custom ASCII Art:** Featuring the unique "Mr.Almm" hacker bunny.
* **Professional Coloring:** Utilizes ANSI escape codes (Red, Blue, and White) for a clean CLI experience.
* **Smart Validation:** Automatically verifies if the MAC address was successfully updated.
* **Lightweight:** Minimal dependencies, works straight out of the box on Linux.

## 🛠️ Requirements
* **OS:** Kali Linux (Recommended) or any Linux distribution.
* **Language:** Python 3.x.
